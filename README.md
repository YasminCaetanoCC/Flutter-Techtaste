<p align="justify">
  O <b>TechTaste</b> é um aplicativo mobile de delivery de comida, inspirado em grandes plataformas do mercado (como o iFood). Este projeto foi desenvolvido como parte do curso de <b>Imersão em Flutter</b>, com o objetivo de aplicar conceitos de construção de interfaces de usuário (UI), navegação e gerenciamento de estado em aplicativos móveis.
</p>

## 📱 Funcionalidades (Features)
* Navegação por categorias de alimentos e restaurantes.
* Visualização detalhada dos pratos e itens do cardápio.
* Adição e remoção de itens no carrinho de compras.
* Interface moderna, fluida e responsiva.

</p>

## 🛠️ Tecnologias Utilizadas
* **Flutter:** Framework principal para construção da interface multiplataforma.
* **Dart:** Linguagem de programação base do projeto.
